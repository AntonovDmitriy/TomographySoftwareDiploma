Digital processing of the projection data

 

Filtered back projection method involves the use of digital filters to received during the experiment or simulation projection data . Without applying them tomogram blurry and unsuitable for the study.

Implementation of digital processing (Fig. 8) is in a class ImageTransormator static method with the signature private static double [] [] filterStatic (double [] [] proj, int views, int scans, int stepsize, String filterName). The input signal is a two-dimensional array of projection data , the scanning conditions and name of the filter to apply to the projection data . In the method of static double [] filter1 (String filtname, int scans, double cutoff) class Utils obtain filtration vector . The outer loop iterates line projection matrix by angle scanning. After selecting the line of projection data and the formation of the filtration of the vector to the line of projection data used direct Fourier transform, the result is multiplied by the filter vector. After that, the inverse Fourier transform [13] and filtered string is written to a new two-dimensional array , the same size as the initial projection data array.
