Введение

 

Данный программный комплекс представляет собой попытку создания современного вычислительного программного обеспечения для томографических исследований. Комплекс состоит из двух частей. Первая часть – демонстрация метода отфильтрованных обратных проекций применительно к компьютерной трансмиссионной томографии. Пользователь может моделировать синограммы (графические представления проекционных данных) и восстанавливать по ним томограммы, используя различные цифровые и цветовые фильтры. Вторая часть –  комплекс для получения проеционных данных и их восстановления для последующего анализа. Программа также непосредственно взаимодействует с базой данных, в которую и попадают проекционные данные после их получения. На данный момент аппаратная составляющая проекта отсутствует и поэтому в комплексе применяется эмулятор работа настоящего томографа. В комплекс встроен разработанный анализатор плотности, который предназначен для количественной оценки спектра плотности образца.  

Проект выполнялся в качестве дипломной работы автора на кафедре Прикладной ядерной физики НИЯУ МИФИ. Проект полностью открытый и бесплатный, любой желающий может свободно использовать его по своему усмотрению в рамках opensource лицензии и изменять без каких-либо опасений.
