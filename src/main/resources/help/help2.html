  import java.util.Arrays;

import basic.labs.Merger;

public class MergeSorting {
	
    public static int[] sort(int[] array, int from, int len) {
    	
        if (len == 0) {
            return new int[0];
        } else if (len == 1) {
        	
            return new int[]{array[from]};
        } else if (len == 2) {
            if (array[from] < array[from + 1]) {
                return new int[]{array[from], array[from + 1]};
            } else {
                return new int[]{array[from + 1], array[from]};
            }
        } else {
            int[] left = sort(array, from, len / 2);
            int[] right = sort(array, from + (len / 2), len - (len / 2));
            return Merger.mergeLab(left, right);
        }
    }
    
    public static void main(String[] args) {
        int[][] data = {
                {9, 7, 0, 3, 5, 2, 6, 1, 4, 8},
                {0, 3, 5, 2, 1, 4},
                {0, 3, 5, 2, 1},
                {0, 3, 5, 2},
                {0, 3, 5},
                {0, 3},
                {0},
                {},
        };
        for (int[] origin : data) {
            System.out.println("origin: " + Arrays.toString(origin));
            int[] sorted = sort(origin, 0, origin.length);
            System.out.println("sorted: " + Arrays.toString(sorted));
            System.out.println();
        }
    }
}

Почему достижение финансовой независимости имеет смысл?

Жизнь человека, это самое дорогое что у него может быть. Правда, многие могут спорить, якобы что они живут ради других. Без сомнений встречаются и такое в нашем мире, и это, наверное, замечательно, но в большинстве случаев те, кто так говорит, просто блефуют. Ибо все таки собственная жизнь для многих является наивысшей ценностью.

А что такое жизнь? Это прежде всего время существования человека. Но, кто-то может возразить заявив, что есть люди, которые все жизнь находятся в летаргическом сне. Можно ли назвать их живыми? С точки зрения биологии несомненно, происходит обмен веществ с окружающей средой, организм обновляется и его существование обладает всеми биологическими признаками живого. С другой точки зрения, такое существования с большим трудом можно назвать жизнью. Является ли время, которое мы проводим во сне жизнью? С второй точки зрения нет. Именно ее и разделяю я. Является ли время, потраченное на нелюбимое занятие жизнью? Нет.

Попробуем немного посчитать. В неделе 24*7=168 часов. Из них 8*7 = 56 часов человек спит. Остается 112 часов. Сколько человек работает? Обычно 8 часов плюс обеденный час, рабочая неделя длится у большинства 5 дней. Итого получаем 112 — 9*5 = 67 часов. Плюс большинство жителей мегаполисов тратят на дорогу как минимум час времени, чтобы добраться до работы Итого минус 2 часа в день на дорогу. 67 — 2*7 = 53 часа. Вы вдумайтесь! Осталось жизни в неделю 53 часа из 168. При в эти часы приходится убираться, готовить еду, ходить по магазинам, оплачивать счета, искать парковочное место?

Так ли не правы люди, которые стараются оптимизировать время оплаты счетов через интернет или пытаются избежать пробок, выбирая общественный транспорт. Посчитаем насколько дольше живет человек, в том смысле, что он занимается только тем, что он по настоящему хочет, не раздумывая о том, на что прокормить себя и свою семью. Начнем отсчет с возраста 23 лет, закончим 60 годами, временем выхода на пенсию. Предположим, у работающего человека 28-дневный отпуск в году и неделя больничного, получается 28+7 = 35 дней в пользу работающего. Это 5 недель.

Посчитаем оставшиеся дни жизни рабочего в год . В году 52 недели. (52 - 5) недель * 53 часа (часов живет в рабочее время) + (24 – 8)*7*5 (живет в течении отпуска и больничных) Итого: 2491 + 560 = 3051 час.

Посчитаем сколько живет в год, у которого есть капитал, который позволяет ему не работать, ради добывания пищи и одежды для своей семьи и себя. 52 Недель * 7 дней * (24 — 8 ) часов = 5824 часа. Получается второй человек живет на 2773 часа больше первого в год. Это 115 дней в год. А именно в 1,91 раза больше.

Так продолжается в течение рассматриваемого периода, который можно описать как время карьеры первого человека.

37 лет. * 3051 час = 112887 часов 37 лет * 5824 часа = 215488 часов

Что ж, почти вдвое? Разница в 102601 час. Это почти на 12 лет больше, а именно 11.74 года. Может быть стоит попытаться дать нашим детям то, что позволит им увеличить жизнь на 12 лет?

Секреты увеличения жизни очень просты, это здоровый образ жизни, голова не плечах и капитал. Вы получаете на 12 лет больше жизни. Причем это больше чем 12 лет. Это 12 лет без сна!!. А если пересчитать со сном, то вуаля. Разница 102601 часов бодрствования и без работы у человека с капиталом.

Это 102601 / (24 – 8) = 6412 дня без работы вместе со сном. 6412 / 365 = 17,5 лет. 17 с половиной лет имеет человек, обладающий капиталом. Может стоит задуматься что для человека значит потребительские кредиты или вообще банковские обязательства? Это почти 18 лет жизни. Много? Я тоже так думаю.
