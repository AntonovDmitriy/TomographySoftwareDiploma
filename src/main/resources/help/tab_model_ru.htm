Вкладка «Модель»

 

     Главное предназначение вкладки «Модель» – моделирование, цифровая фильтрация и визуализация проекционных данных, а также восстановление по ним исходного изображения по методу отфильтрованных обратных проекций. При старте пользователь может видеть следующее окно.

 

Рисунок 11. Внешний вид модуля моделирования комплекса.

 Примеры изображений для моделирования встроены в комплекс для простоты демонстрации, список моделей находится в левом верхнем углу. 

Также доступны для загрузки любые сторонние изображения в форматах bmp, jpeg и png. Для того, чтобы выбрать стороннее изображение достаточно нажать кнопку «Открыть файл» и выбрать графический файл. При загрузке (рис. 13) происходит преобразование изображений в восьми-битный формат и отображение происходит  в черно-белом цвете.

 

Рисунок 13. Открытие встроенного в комплекс изображения для  моделирования синограммы.

В левой стороне окна располагаются три области ввода параметров моделирования: количество сканирований (число смещений источника относительно образца), шаг сканирования (шаг поворота системы источник-детектор в градусах), размер реконструкции (размер квадратного изображения реконструкции в пикселах). Необходимо вводить целочисленные значения параметров, иначе программа выдаст предупреждение о некорректности данных.

В правой графической области при нажатии кнопки «Синограмма» отображается синограмма исходного изображения (рис. 15), которая представляет собой, по сути, графическую интерпретацию проекционных данных, при этом двумерный массив проекционных данных находится в памяти программы. После получения проекционных данных, синограмму можно сохранить в память компьютера нажатием кнопки «Сохранить синограмму», которая станет активной. Процесс расчета отображается в бегущей полосе загрузки. (рис. 14).

Рисунок 15. Отображение синограммы моделируемого изображения.

 

 

При нажатии кнопки «Реконструкция» происходит восстановление томограммы по проекционным данным, полученным на предыдущем шаге. Необходимо применять цифровые фильтры при реконструкции изображения, так как иначе изображение получается нечетким (рис.16).

Фильтрация задается активацией чек-пойнта «Фильтрация», при этом ниже появляются доступные цифровые фильтры. При реконструкции необходимо помнить, что параметры моделирования должны сохраняться постоянными за исключением размера реконструкции, который при моделировании проекционных данных не играет никакой роли, иначе это приведет к ошибочной ситуации, о чем сообщит программа. В правом окне синограмма меняется на томограмму (рис. 17), причем при превышении размера графического поля, появляются полосы прокрутки, этим самым можно обеспечить зумирование для детального исследования изображения.

 

 

Рисунок 16. Пример реконструкции изображения без цифровой фильтрации.

 

Рисунок 17. Реконструкция смоделированных проекционных данных с применением фильтра Ганна.

 

При активизации цветовой фильтрации открывается панель выбора цветовой коррекции томограммы. Также размемеры  трех графических областей можно свободно изменять мышкой. При этом реализован многооконный интерфейс, который позволяет при двойном нажатии на любом изображение, будь то синограмма или реконструкция, показать его в отдельном окне.
