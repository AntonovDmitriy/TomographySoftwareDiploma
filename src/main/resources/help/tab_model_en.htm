Tab "Model"

 

        The main purpose of the tab "Model" - modeling, digital filtering and visualization of the projection data, as well as restoring them to the original image by the method of filtered back projection. At startup, the user can see the following window.

 

Figure 11. Appearance module modeling complex.

 

 Sample images for modeling complex built in for easy demonstration of models in the upper left corner.

Also available for download any third-party image formats: bmp, jpeg and png. In order to select the side of the image just click "Open File" and select the image file. When loading (Fig. 13) converts images into eight-bit format and display comes in black and white. 

 

Figure 13. Opening embedded in complex images to simulate sinogram.

 

On the left side of the window there are three input field simulation parameters : the number of scans ( the number of displacements source relative to the sample ) , the scanning step (step rotation of the source- detector degrees) , the size of the reconstruction ( reconstruction image size square pixels) . Must enter integer values ​​, otherwise the program will warn you about incorrect data.

In the graphics area, right by pressing the "sonogram" sinogram displays the original image (Fig. 15) , which is essentially a graphical interpretation of the projection data , with the two-dimensional array of projection data is in the program memory . After receiving the projection data sinogram can be stored in computer memory by pressing the “Save sinogram", which will become active. The process of calculation is shown in a traveling band download. ( Fig. 14). 

Figure 14. Sinogram computation process.

Figure 15. Displaying sinogram simulated images.

 

Pressing the "Reconstruction" is a restoration of tomograms by projection data obtained in the previous step . Necessary to apply digital filters in image reconstruction , since otherwise the image turns fuzzy ( Fig. 16 ) .

Filtration is defined by the drop down list "filtering" . When reconstruction is necessary to remember that the simulation parameters must be kept constant except for the size of reconstruction which the simulation projection data plays no role , otherwise it will lead to an error condition , and an error program . In the right window appears tomogram (Fig. 17), and in excess of the size of the graphic field , scroll bars appear .
