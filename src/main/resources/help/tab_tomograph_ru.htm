Вкладка Томограф

 

На вкладке «Томограф» (рис. 21) располагаются инструменты управления томографом. В настоящее время в код встроен эмулятор работы томографа, но имеется возможность получать по аппаратному интерфейсу данные с реального устройства. Начальные условия представляют собой поля числовых данных для конкретного томографического исследования образца. Это шаг исследования вдоль образца, шаг поворотного механизма, шаг смещения источника относительно образца, размер реконструкции. Возможность изменения параметров исследования реализована для более гибкого применения комплекса для возможности его применения к задачам разной степени точности.

Производится соединение с удаленной базой данных проекционных данных и отображение списка данных в таблице. Сейчас используется база данных MySql расположенная на сервере сервиса OpenShift. Базу данных можно изменить, подправив соответствующие параметры в файлах настроек (доступно только для установленной версии программы). При соединении с базой данных используется шифрование.

 

Рисунок 21. Вид вкладки «Томограф» в начале работы с программой.

 

Кнопка «Старт» запускает программу управления двигателями и системой сбора данных. Перед началом программы выводится диалог (рис. 22),  в котором пользователю предлагается ввести название файла проекционных данных, под которым он сохранится в базу данных. Процесс получения данных и их записи сопровождается бегающей полоской, которая означает работу программы, в процессе которой можно перейти на вкладку «Модель» без каких-либо последствий для процесса вычисления.

После завершения работы программы в базу данных сохраняется набор проекционных данных, реконструировать который можно нажатием клавиши «Реконструкция». В комментариях удобно записывать параметры сканирования, которые должны точно соответствовать параметрам реконструкции для верной работы алгоритма. При выборе набора проекционных данных программа спрашивает цифровой фильтр, который необходимо применять в процессе построения томограммы.

 

Рисунок 22. Окно ввода описания набора проекционных данных после    нажатия кнопки «Старт».

  

После реконструкции серия изображений контролируемого образца появляется в графической области в центре вкладки «Томограф». При движении ползунка (рис. 24) можно наблюдать изменение изображения поперечного сечения образца вдоль его длины, причем каждую полученную реконструкцию можно сохранить, нажав кнопку на панели инструментов «Сохранить реконструкцию».

 

Рисунок 24. Изменение вида томограммы образца вдоль его длины.

 

         При передвижении вдоль длины образца возможно применение разных цветовых фильтров, а также открытие каждого изображения в отдельном окне и использовать зумирование (рис. 25).

 

Рисунок 25. Применение цветовых фильтров на вкладке «Томограф».

Цветовая фильтрация также доступна в модуле «Томограф», для этого нужно активизировать одну из опций выпадающего списка «Цветовой фильтр»  и выбрать соответствующий фильтр из появившихся вариантов.
