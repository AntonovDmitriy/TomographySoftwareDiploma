Tab Tomograph

 

On the tab " tomograph " (Fig. 21) located tomograph management tools . Currently, the code is embedded emulator working scanner , but it is possible to receive data from the hardware interface of a real device . The initial conditions are fields of numerical data for a particular sample tomography . It is a step along the research sample , step ratchet with bias source relative to the sample size reconstruction. Ability to change the parameters of the study implemented for a more flexible use of the complex for its application to problems of varying degrees of precision.

The connection is made to a remote database projection data and display a list of data in a table . Now used MySql database located on a server service OpenShift. Database can be changed by modifying the settings in the configuration files (available only for the version of the program). When the database connection uses encryption.

.

Figure 21. View tab " tomograph " at the beginning of the program.

 

"Start" button starts the engine control and data acquisition system. Before the program dialog is displayed (Figure 22) , in which the user is prompted to enter the file name of the projection data , under which it will be saved to the database. Process of obtaining data and writing accompanied by a running strip , which means the program , during which you can go to the tab "Model" without any consequences for the calculation process .

After the program is stored in a database set of projection data , which can be reconstructed by pressing the "Reconstruction" . The commentary is convenient to write the scan settings , which should match the parameters for the faithful reconstruction of the algorithm. When choosing a set of projection data, the program asks digital filter, which must be used during the construction of the tomogram.

 

Figure 22. Input window describe a set of projection data after pressing "Start".

After the reconstruction of a series of images controlled sample appears in the graphics area in the middle of the tab " tomograph ." When moving the slider (Fig. 24) can be seen changing cross-sectional images of the sample along its length , and each received reconstruction can be saved by clicking on the toolbar " Save reconstruction".

Figure 24. Changing the type of tomograms of the sample along its length.

When traveling along the length of the sample is possible to use different color filters , as well as opening each image in a separate window and use the zoom ( Fig. 25).

Figure 25. Apply color filters on the " Imager ".

Color filtering is also available in the module " tomograph " , it needs to activate one of the options drop-down list " Color Filter " and select the appropriate filter options emerged.
