Цифровая обработка проекционных данных

 

Метод отфильтрованных обратных проекций предполагает применение цифровых фильтров к полученным в ходе эксперимента или  моделирования проекционным данным. Без их применения томограмма получается размытой и непригодной для изучения.

         Реализация цифровой обработки (рис. 8) находится в классе ImageTransormator в статическом методе с сигнатурой private static double[][] filterStatic(double[][] proj, int views, int scans, int stepsize, String filterName). На вход поступает двумерный массив проекционных данных, условия сканирования и имя фильтра, который необходимо применить к проекционным данным. В методе static double[] filter1(String filtname, int scans, double cutoff) класса Utils получаем фильтрационный вектор. Внешний цикл перебирает строки проекционной матрицы по углу сканирования. После выбора строки проекционных данных и формирования фильтрационного вектора, к строке проекционных данных применяется прямое преобразование Фурье, полученный результат умножается на фильтрационный вектор. После этого производится обратное преобразование Фурье [13] и отфильтрованная строка записывается в новый двумерный массив, такой же по размеру как и начальный массив проекционных данных.

 

         В комплексе присутствуют следующие цифровые фильтры [14]:

 

1)   Ramp

 

 

2)   Shepp-Logan

 

3)   Hamming

 

 

4)   Hann

 

 

5)   Сosine
